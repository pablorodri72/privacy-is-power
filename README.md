# Privacy is Power — Proyecto de lectura y creatividad

Este proyecto lo he hecho para la asignatura de Protección de Datos y Ciberseguridad.

La idea era leer un libro relacionado con privacidad o tecnología (en mi caso *Privacy is Power* de Carissa Véliz) y montar una web donde se vea que lo has entendido, pero de forma más creativa.

He intentado que la página tenga un estilo un poco diferente, inspirándome en los DVDs antiguos, con secciones, navegación lateral y un mapa interactivo de autores.

## Ver la web

https://pablorodri72.github.io/privacy-is-power/

## Qué hay en la web

* Un resumen del libro con las ideas principales
* Citas importantes con pequeñas explicaciones
* Mi opinión personal (qué me convence, qué no, etc.)
* Una parte de crítica donde discuto algunos argumentos
* Un mapa interactivo con autores relacionados (Foucault, Zuboff, Snowden…)

## Tecnologías

He usado cosas bastante básicas:

* HTML
* CSS
* JavaScript

## Comentario personal

Más que hacer un resumen sin más, he intentado entender bien el libro y conectar sus ideas con cosas actuales, sobre todo con temas de ciberseguridad y cómo usamos nuestros datos hoy en día.

También me he apoyado en sistemas de IA para ayudarme con el código y la estructura.
